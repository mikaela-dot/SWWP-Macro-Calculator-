# Sérenité — Macro Tracker PWA

Personal macro tracking app by MDeats Wellness.

## Deploy to GitHub Pages

1. Create a new GitHub repository (public)
2. Upload all files from this folder
3. Go to Settings → Pages → Source: Deploy from branch → main → / (root)
4. Wait ~2 minutes, then visit `https://yourusername.github.io/repo-name`

## Add to Phone Home Screen

**iPhone (Safari):**
1. Open your GitHub Pages URL in Safari
2. Tap the Share icon (square with arrow)
3. Scroll down → tap "Add to Home Screen"
4. Tap Add — the Sérenité icon appears on your home screen

**Android (Chrome):**
1. Open your GitHub Pages URL in Chrome
2. Tap the 3-dot menu
3. Tap "Add to Home Screen"
4. Tap Add

## Files

- `index.html` — the full app
- `manifest.json` — PWA configuration (enables home screen install)
- `sw.js` — service worker (enables offline use)
- `icon-*.png` — app icons for all device sizes
- `apple-touch-icon.png` — iPhone home screen icon

## Features

- 📊 Live macro dashboard with ring progress indicators
- 🔍 198-food searchable database
- 📷 AI nutrition label scanner (Camera or Gallery)
- 📅 Daily log with meal grouping (Breakfast/Lunch/Afternoon/Dinner)
- 📈 Weekly review with 7-day history
- 💾 Automatic daily save — data persists across sessions
- 🌿 MB protocol rules and targets built in
- 💊 Cortisol recovery phase targets (protein 171g, carbs by day type)
